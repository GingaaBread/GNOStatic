# Keywords

Keywords are reserved expressions that **cannot** be used as identifiers.
This is a complete list of keywords in GNO sorted lexicographically:

A:

- abstract
- and
- any
- as
- at

B:

- boolean
- break

C:

- case
- catch
- char
- class

D:

- default
- do
- double

E:

- else
- empty
- enum
- equals
- entity
- event
- exists

F:

- false
- first
- for
- foreach

G:

- get

I:

- if
- implemented
- in
- int
- interface
- is
- it

L:

- last

N:

- not
- null

M:

- maybe
- mod

O:

- of
- or
- other
- overridden

S:

- set
- single
- static
- service
- string
- switch

T:

- then
- throw
- true

W:

- while
- with
